# 假期星图 ⭐

> 一个优雅的假期与工作倒计时 uTools 插件，让时间可视化，让期待更美好。

当前版本：v2.1.1

## ✨ 特色功能

### 🏃 下班倒计时
- 实时显示距离下班的剩余时间
- 支持自定义工作时间和工作日
- 渐变卡片设计，一目了然

### 🎉 下班礼花
- 下班时刻自动放礼花庆祝
- 支持多显示器全屏覆盖
- 双击下班卡片手动触发彩蛋
- 可配置颜色方案和强度

### 🎊 节日倒计时
- 自动同步中国法定节假日（元旦、春节、清明、劳动节、端午、中秋、国庆）
- 每个节日都有独特的主题配色
- 国庆卡片特别添加五星红旗图案，符合国旗法标准
- 显示剩余天数，提前规划假期

### ✨ 自定义倒计时
- 添加个人重要日期（生日、纪念日、考试等）
- 支持备注说明
- 灵活管理，随时添加删除

### 🎨 精美设计
- 现代化渐变配色，视觉舒适
- 大号时钟显示当前时间和农历
- 卡片 hover 动效，交互流畅
- 完整适配深色模式

## 🚀 快速开始

### 安装
在 uTools 插件中心搜索"假期星图"或"倒计时"即可安装

### 使用
1. **查看倒计时**：在 uTools 中输入 `假期星图`、`倒计时` 或 `holiday`
2. **设置参数**：点击右上角"设置"按钮，或输入 `假期设置`

## 📖 功能说明

### 工作时间设置
1. 进入设置页面
2. 设置上班时间和下班时间
3. 选择工作日（支持多选）
4. 点击"保存工作时间"

### 节假日同步
1. 进入设置页面
2. 点击"同步节假日"按钮
3. 自动从官方数据源获取最新节假日信息
4. 支持清空缓存重新同步

### 自定义倒计时
1. 进入设置页面
2. 填写名称、日期和备注（可选）
3. 点击"添加倒计时"
4. 在主页面即可查看

## 🎨 节日配色说明

每个节日都有精心设计的主题色：
- **元旦**：金橙色渐变 - 新年喜庆
- **春节**：中国红渐变 - 传统喜庆
- **清明**：青灰蓝渐变 - 肃穆追思
- **端午**：绿色渐变 - 粽叶龙舟
- **中秋**：金黄色渐变 - 月饼圆月
- **国庆**：国旗红 + 五星图案 - 爱国情怀

## 🛠️ 技术栈

- **框架**：Vue 3 + Vite
- **状态管理**：Pinia
- **日期处理**：Day.js + dayjs-plugin-lunar
- **日历解析**：ical.js
- **UI 组件**：Naive UI
- **平台**：uTools

```bash
# 构建
pnpm build
```
